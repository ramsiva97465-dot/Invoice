# Project Brain Runtime

This repository contains the **Project Brain** runtime – a structured set of documentation, standards, and task tracking files to support project development.

## Folder Structure

- `system/` – Core system documentation (overview, planner, workflow).
- `memory/` – Knowledge base about the frontend, backend, database, and API.
- `standards/` – Coding and design standards for React, TypeScript, databases, and UI.
- `tasks/` – Active, completed, and changelog task tracking.
- `cache/` – Cached context and recent files for quick reference.

Each file contains a brief placeholder that can be expanded by the team.

*This runtime is initialized and does not modify any existing application source code.*
