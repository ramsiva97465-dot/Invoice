# Project Brain System Overview

This document outlines the overall architecture and purpose of the **Project Brain** runtime. It serves as a high‑level entry point for developers to understand the folder hierarchy, key components, and how to extend the runtime.

- **system/** – Core configuration and planning files.
- **memory/** – Documentation of domain‑specific knowledge areas.
- **standards/** – Coding standards and best‑practice guidelines.
- **tasks/** – Tracking active and completed tasks.
- **cache/** – Cached context for quick reference.

> This file is intentionally concise; detailed sections are provided in the respective sub‑folders.
